Meet App

This app is a a serverless, progressive web application (PWA) with React using a test-driven development (TDD) technique. The application uses the Google Calendar API to fetch upcoming events and uses the OAuth2 authentication flow to authenticate visitors. Users can discover events from various cities and data is displayed using tools from Recharts.
