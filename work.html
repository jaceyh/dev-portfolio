Movies App (React)

For this project, I first created the documentary movie database in PostgreSQL to become familiar with relational database systems. The final version of the API was created using MongoDB, Express, and Node.js and endpoints were tested with Postman. The API is deployed through Heroku. For the frontend, I created a single page application using React.js to complete the MERN stack. The live app is deployed through Netlify.

Meet App

This app is a a serverless, progressive web application (PWA) built with React using a test-driven development (TDD) technique. The application uses the Google Calendar API to fetch upcoming events and uses the OAuth2 authentication flow to authenticate visitors. Users can discover events from various cities and data is displayed using tools from Recharts.
